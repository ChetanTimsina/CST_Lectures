##  How to use

1. Browse this repository and click the folder/course you want.  
2. On the course repository page, you can:  
   - Clone the repo:  
     ```bash
     git clone <course-repo-link>
     ```  
   - Download ZIP for offline use.  

3. Each course is **independent** — you can clone only the courses you need without downloading the entire CST Lectures repo.
